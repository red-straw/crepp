# Pointer Dungeon

## GPT 활용

본 프로젝트는 GPT를 활용하여 게임 기획, 프로그래밍 및 학습 콘텐츠 제작을 지원받아 개발되었습니다.

## 게임 소개

Pointer Dungeon은 C 언어와 Raylib를 활용하여 제작한 교육용 게임입니다. 플레이어는 던전을 탐험하며 C 언어 문제를 해결하고, 이를 통해 변수, 포인터, 배열 등 C 언어의 핵심 개념을 자연스럽게 학습할 수 있습니다.

## 주요 기능

* C 언어 및 Raylib 기반 개발
* C 언어 퀴즈 풀이 시스템
* 점수 및 체력 시스템
* 게임을 통한 C 언어 학습
